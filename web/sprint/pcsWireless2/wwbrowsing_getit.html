Getting the Sprint PCS Wireless Web Browser
To use the Sprint PCS Wireless Web Browser, you will need a phone that is wireless Internet  ready. No sign up is required to use the Wireless Web Browser. However, our Wireless Web Option allows you to use your voice/data minutes for wireless web browsing, which is more economical than the default per minute rate.
Note to Existing Subscribers
Check the phone list to see if your phone is wireless Internet-ready. Some phones may be upgraded, or you may want to consider purchasing a new wireless Internet ready Sprint PCS Phone. Go here to get the latest version of Sprint PCS Dialer software.
