Troubleshooting
If you're having a problem using the Sprint PCS Web Browser, you may be able to find a solution here. If your problem isn't listed here, or if the suggested solution doesn't help, let us know. Sprint PCS provides world class Customer Care anytime, anywhere. Just dial *2 on your Sprint PCS Phone or dial 1-888-211-4727 to connect to a Sprint PCS Customer Care representative.
  1. If you have trouble accessing a Wireless Web content provider.
  2. If you have trouble with error messages when you try to connect.
  3. If you are not sure if your Sprint PCS Phone is browser-ready.
  1. If you have trouble accessing a Wireless Web content provider.
    There are several possibilities: the service is no longer available, the URL has changed, or the service's response time is too slow. Try again later, or contact the provider of the service.

    2. Return to Top

  2. If you have trouble with error messages when you try to connect.
    The provider's site may be busy or down for repairs. Try again later. If problems persist, contact the provider of the service.

    3. Return to Top

  3. If you are not sure if your Sprint PCS Phone is browser-ready.
    Refer to the list of wireless Internet ready phones. Or, simply press the Menu key on your phone and look for the listing for "MiniBrowser." If you see the MiniBrowser listing, your phone is wireless Internet ready.

    4. Return to Top
