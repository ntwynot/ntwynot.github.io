Getting Wireless Web Updates from Yahoo!
All Sprint PCS Phones support Wireless Web Messaging, which allows you to receive Wireless Web Updates from Yahoo! To shop for a Sprint PCS Phone, visit our online store.
Signing Up
To receive Wireless Web Updates, you must be registered for MyYahoo! and either a Wireless Web service option or the Wireless Web Messaging option.
  • If you don't have any of these plan options, you'll be asked to choose one when signing up for Wireless Web Updates.
  • If you already have one of these plan options, then you're ready to sign up for Wireless Web Updates. Subscribers to the 30 Sprint PCS Wireless Web Messaging Option with 30 messages may upgrade their plan during the sign-up process.
  • If you don't know what plan you have, don't worry. We'll remind you when you sign up for Wireless Web Updates.
During the sign-up, you can register for MyYahoo! Once you've signed up, you can then customize your Wireless Web Updates. That's it! You'll start receiving Wireless Web Updates within 24 hours after signing up.
