< Back < Print Advertising — Real Estate Digital

Print related graphic design for Real Estate Digital. Includes homes for sale flyers information that can be generated as an email and/or as a printable flyer. These designs are built into the rDesk (internal back-end admin) platform for Agents. Designs are composed by using a combination of data-based widget tokens, within an html framework, consisting of css and images to style them. The code behind the tokens within the system would load a photo of the agent, property photos, and the property information.
